P.P.S. 🥚 This page has rich data

Every book has structured HTML attributes you can query in the browser console:

  • data-year — publication year
  • data-starred — favourite
  • data-status — "read" or "reading"
  • data-caldecott-medal / data-caldecott-honor — Caldecott award year
  • data-cbca-picture-book — CBCA Picture Book of the Year award year
  • data-nyt-illustrated — NYT Best Illustrated year

Try these in your browser console:

// All starred books
document.querySelectorAll('li[data-starred]')

// Books from the 1990s
document.querySelectorAll('li[data-year^="199"]')

// Caldecott winners
document.querySelectorAll('li[data-caldecott-medal]')

See the README for more queries.
